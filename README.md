# Puzzdle - Battle Arena

Puzzdle, oyuncuların gerçek zamanlı olarak yapboz çözerek birbiriyle rekabet edebildiği, sesli sohbet ve yapay zeka entegrasyonuna sahip çok oyunculu bir web platformudur.

## Özellikler
- **Battle Arena (Düello Odaları):** Arkadaşlarınızla aynı anda yarışın.
- **Gerçek Zamanlı İletişim:** Socket.io entegrasyonu ile anlık durum güncellemeleri ve sesli sohbet.
- **AI Studio:** Yazdığınız promptlar (metinler) üzerinden saniyeler içinde benzersiz yapboz görselleri üreten yapay zeka entegrasyonu.
- **Liderlik Tablosu & Profil:** Skor takibi ve kullanıcı istatistikleri.
- **Admin Paneli:** Kullanıcı ve şablon yönetimi.

## Teknolojiler
- Frontend: HTML, CSS, JavaScript
- Backend/Real-time: Node.js, Socket.io
- Yapay Zeka Entegrasyonları (Görsel Üretimi)

## Kurulum
1. Projeyi klonlayın.
2. Backend dizininde `npm install` komutu ile bağımlılıkları yükleyin.
3. `npm start` ile sunucuyu başlatıp, tarayıcınızda `index.html` üzerinden erişin.
